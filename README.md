# indian-stock-market-analytics
Python-based Indian Stock Market Analytics Dashboard with live data, technical indicators, stock comparison, and portfolio simulation.
# Indian Stock Market Analytics Dashboard
![image alt](https://github.com/Nitrogen345/indian-stock-market-analytics/blob/16796d480537a2edd4fec15c91d1ab95d9643cbf/Dashboard.png)

An interactive **stock market analytics dashboard** built using Python.  
This project allows users to analyze Indian stock market data with technical indicators, visualizations, and portfolio insights.

## Overview

This dashboard fetches live stock market data and provides analytical tools to help understand stock performance, trends, and volatility.  
It includes interactive charts, technical indicators, and comparison tools for analyzing multiple stocks.

## Features

- Live stock data using Yahoo Finance
- Candlestick price chart
- Technical indicators (SMA20, SMA50, RSI)
- Daily return and volatility analysis
- Stock comparison with multiple companies
- Portfolio investment simulation
- Interactive charts using Plotly

## Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- yfinance

## How It Works

1. The application fetches live stock data using the **yfinance** library.
2. Data is processed using **Pandas** and **NumPy**.
3. Technical indicators such as moving averages and RSI are calculated.
4. Interactive charts are generated using **Plotly**.
5. The dashboard interface is built using **Streamlit**.
